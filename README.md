## 🛠️ 使用指南

### 📖 阅读Markdown文档
- 推荐使用支持数学公式的Markdown编辑器
- 如：Typora、Mark Text、VS Code + Markdown Preview Enhanced

### 🎨 查看架构图表

**多格式支持**:
- **`.drawio` 文件**: 原始可编辑格式，支持动画效果
- **`.svg` 文件**: 矢量图格式，适合网页展示和缩放
- **`.png` 文件**: 位图格式，适合文档嵌入和演示

**在线查看**:
1. 访问 [diagrams.net](https://app.diagrams.net/)
2. 选择 "Open Existing Diagram"
3. 上传对应的 `.drawio` 文件
4. 点击播放按钮查看动画效果

**本地查看**:
1. **动画版本**: 下载 [draw.io Desktop](https://github.com/jgraph/drawio-desktop/releases)，打开 `.drawio` 文件
2. **静态版本**: 直接在浏览器或图片查看器中打开 `.svg` 或 `.png` 文件
3. **文档嵌入**: 在Markdown文档中直接引用 `.svg` 或 `.png` 文件

**格式选择建议**:
- **演示汇报**: 使用 `.png` 格式，兼容性最好
- **网页展示**: 使用 `.svg` 格式，支持无损缩放
- **交互学习**: 使用 `.drawio` 格式，可编辑和查看动画
- **文档编写**: 根据需要选择 `.svg`（矢量）或 `.png`（位图）
